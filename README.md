# Dream-App
Image generator using DALL·E. Vanilla JavaScript app with Vite. 

Head to [DALL·E Image Generation Docs](https://platform.openai.com/docs/guides/images) to log in and create an API key.

Run the backend
`node server.js`

Start the frontend
`npm run dev`
